PASO 1

Descarga la aplicación del lector en tu movil Android e iOS

iOS - iPhone

Se utiliza la aplicación Spike. Esta aplicación se conecta directamente con el Miaomiao y recibe lecturas de glucosa cada 5 minutos. Para descargarla debes seugir los sigueintes pasos:


1. Accede a su página web oficial y solicita su descarga.

2. Rececibirás un mail con las instrucciones de descarga y pasos a seguir para su instalación (debes abrir este mail desde el iPhone).


Instalar Spike

Andorid

Se utiliza la aplicación xDrip+. Esta aplicación se conecta directamente con el Miaomiao y recibe lecturas de glucosa cada 5 minutos. Para descargarla debes seugir los sigueintes pasos:


1. Accede al enlace de descarga que se muestra debajo de este texto (debes acceder desde el movil Android).

2. Guarda la descarga (termina en .apk) y posteriormemte ve a la carpeta de descargar de tu movil.

3. Pulsa sobre la descarga y dale click a instalar.
